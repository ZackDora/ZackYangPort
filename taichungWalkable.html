步行臺中

人本交通推廣網站

  • 角色：前端開發 / 內容規劃
  • 類型：課程專案
  • 時間：2024
  • 網站： taichungwalkable.netlify.app
  • GitHub： 專案原始碼

專案簡介

《步行臺中》是一個以「人本交通」為核心的推廣型網站， 目標是透過整理概念、案例與觀察，讓更多人理解以行人為中心的交通設計， 並將相關想法應用於臺中的城市環境。

動機與背景

初到臺灣時，我對整體交通秩序感到驚訝與敬佩，但在實際生活與學習過程中，也逐漸意識到行人環境與人本交通仍存在許多值得改善的空間。

透過 YouTube 與相關資料接觸到城市規劃與人本交通的概念後， 我開始思考：是否能用一個網站，整理這些觀點， 並以臺中為例，作為分享與討論的起點。

內容規劃與方向

專案初期的構想較為宏觀，試圖全面分析臺中交通問題。 在多次回饋與修正後，內容逐步聚焦， 最終選擇以三個具體議題作為核心：

  • 人行道設計
  • 交叉路口安全
  • 停車與空間分配

網站本身定位為「觀念分享與意見交流的平台」， 而更完整的分析則整理為獨立報告供讀者下載閱讀。

使用技術

  • HTML / CSS
  • JavaScript（導覽選單、互動）
  • Eleventy（靜態網站生成器）
  • Netlify（部署）
  • Netlify CMS（內容管理）

製作過程

本專案從零開始建構網站結構與樣式， 並隨著內容定位的轉變，多次調整網站架構， 最終採用類部落格形式，提升內容維護與延展性。

在完成基本的 HTML、CSS 與 JavaScript 後， 導入 Eleventy 將網站轉為可擴充的靜態網站， 並透過 GitHub 與 Netlify 進行版本控管與部署。

反思與未來計劃

這個專案讓我同時深化了對人本交通議題的理解， 也實際鍛鍊了前端開發與網站架構規劃的能力。

未來計劃持續優化網站功能， 包含互動機制、內容分類與技術層面的升級， 並逐步發布更多相關文章。
