慈善組織審查網

一個具有社會價值的課程專題網站

角色
全程負責 - 設計, 前端開發
使用技術
HTML, CSS
類型
課程專案
年份
2023

在 2023 年，我修習了一門名為「商用英文」的課程。教授希望我們能透過小組專案，設計並製作一項具有社會價值或經濟價值的產品。當時，我決定以慈善組織為主題，蒐集並評估我們所熟悉的慈善機構。

由於那段時間我正好想練習自己自學的網站開發技能，因此向小組成員提出製作網站的想法。該網站使用簡單的 HTML 與 CSS 製作，整體概念類似一個評分平台，讓使用者可以對不同的慈善組織進行整理與評價。

學習到的技能

透過製作這個網站，我得以實際練習自己的 HTML 與 CSS 技能，特別是在 CSS 的部分。我在頁面設計的各個區塊中，練習使用 Flexbox 與 Grid 進行版面配置，同時也逐步理解並應用一些基本的網站設計邏輯。

在專案報告當天，小組因為這個網站獲得教授的肯定，也引起許多同學的興趣，紛紛詢問網站的製作過程。透過這次專案，我不僅獲得了實作技能上的鍛鍊，也從中感受到極大的成就感。
